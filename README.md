# Ai + Coding Future Academy

Frontend quiz app for Ai + Coding Future Academy courses.

## About
Interactive quiz platform for "Calculus in Action" course.  
Built separately from RIZOME/CreatorsKit. Part of Ai + Coding Future Academy learning suite.

## Tech Stack
- **Frontend**: Next.js 14 + React + Tailwind CSS
- **Database**: Supabase
- **Hosting**: Vercel
- **Course ID**: 5c7383d3-c9c6-464b-97af-618aafa2b8c1

## Features
- Loads 30 quiz questions from Supabase
- One question at a time with instant green/red feedback
- Score tracking + progress bar
- Mobile responsive + fast on Vercel
- Branded for Ai + Coding Future Academy

## Setup for Vercel
1. Connect this repo to Vercel
2. Add environment variables:
   - NEXT_PUBLIC_SUPABASE_URL
   - NEXT_PUBLIC_SUPABASE_ANON_KEY
3. Deploy from `main` branch
